# Data Types

## Rust - Scala Types 

스칼라는 하나의 값으로 표현되는 타입들을 말한다. 

Rust는 정수형, 부동소수점, boolean, 문자의 네 가지 스칼라 타입을 보유하고 있다.

Rust의 정수 타입 

| Length | Signed | Unsigned |
| ------ | ------ | -------- |
| 8bit   | i8     | u8       |
| 16bit  | i16    | u16      |
| 32bit  | i32    | u32      |
| 64bit  | i64    | u64      |
| arch   | isize  | usize    |

Rust의 정수 리터럴

| Number Literals | Example     |
| --------------- | ----------- |
| Decimal         | 98_222      |
| Hex             | 0xff        |
| Octal           | 0o77        |
| Bianry          | 0b1111_0000 |
| Byte            | b'A'        |
